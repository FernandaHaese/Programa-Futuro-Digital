O mixin permite quase criar uma função dentro do scss

Assim posso criar estilos de uma forma mais flexível

Posso ter um mixin com argumentos

Esses argumentos tem um default

Ou seja, um valor padrão

Com herança, posso fazer uma classe herdar as propriedades de outra classe

Cuidado, que as classes filhos podem sobreescrever as propriedades da classe pai
