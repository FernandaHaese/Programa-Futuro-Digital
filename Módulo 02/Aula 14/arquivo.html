Sass

Um sistema Sass (Syntactically Awesome Style Sheets) é um pré-processador CSS que adiciona funcionalidades avançadas ao CSS, como variáveis, aninhamento, mixins (funções) e herança, para tornar o código mais modular, reutilizável e fácil de manter. O código Sass é escrito numa linguagem que é convertida para CSS padrão para que possa ser lido e interpretado pelos navegadores web.
