NPM

O Node Package Manager é utilizado para instalar e gerenciar bibliotecas externas. Por exemplo, para baixar uma biblioteca com o recurso de Sincronizar Prompts, você pode executar no terminal: npm install prompt-syn. Se não executar no terminal de seu editor de código, execute no prompt de comando de seu sistema.

Módulos

Em nosso código, é importante seguir boas práticas e manter nosso conteúdo organizado. Para isso, é recomendado criar módulos para cada funcionalidade nova de seu código.
